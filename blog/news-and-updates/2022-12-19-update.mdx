## Motoko: インタラクティブなコード・スニペットをMediumのストーリーやブログ記事に直接埋め込むことができます！

![Header image](../_assets/medium.png)

実行可能な[Motoko コード・スニペットを](https://embed.smartcontracts.org/) [Medium](https://medium.com/)
ソーシャル・パブリッシング・プラットフォームに直接埋め込む新しいコラボレーションを発表できることを嬉しく思います。
 Motoko 'のドキュメントをオーバーホールする継続的な取り組みの一環として、IC
コミュニティの誰もがインターネット上でインタラクティブなチュートリアルやコード例を簡単に作成できるようにしたいと考えました。
 投稿の全文は[こちらから](https://forum.dfinity.org/t/embed-motoko-interactive-code-snippets-directly-in-your-medium-story-or-blog-post/17433)ご覧いただけます。

## React Native \<-\>についてInternet Computer

![Header image](../_assets/react-native.png)

React Nativeは、開発者が
JavaScriptとReactを使用してネイティブモバイルアプリケーションを構築できる人気のオープンソースフレームワークです。React Nativeを使用すると、開発者は、各プラットフォーム用に別々のコードベースを維持する必要なく、ネイティブアプリのように感じられ、
、高品質のクロスプラットフォームアプリを作成できます。

これまでReact Nativeでは、
JavaScriptで暗号計算を実行するために必要なAPIのサポートが限られていました。しかし、BigIntサポートのリリースにより、
 Internet Computer と直接通信するモバイルアプリの構築とデプロイが可能になりました！

ここでは、agent-js (0.15)を使用した、パッチ不要のスターターテンプレートを用意しています！チェックして、
ご意見をお聞かせください！あなたが作るものを見るのを楽しみにしています。

## 今週のアプリFig

![Header image](../_assets/fig.png)
Fig は、CLI ツールやスクリプトに視覚的なオートコンプリートを追加できるアプリケーションです。私たちチームの何人かは、数週間前から
を使っていますが、生産性が向上しています。ターミナルのためのOSのようなものです。

以下は、私たちが気に入っている点のトップ3です：

CLI のドロップダウンコマンドメニュー
エイリアスと個人用ユーティリティスクリプトの統合
`fig ai` - 自然言語を bash コマンドに翻訳。

などなど。

Figのダウンロードは[こちらから](https://fig.io/)。

## React Native <-> Internet Computer
![Header image](../_assets/react-native.png)

React Native is a popular open-source framework that allows developers to build native mobile applications using
JavaScript and React. With React Native, developers can create high-quality, cross-platform apps that feel and perform
like native apps, without the need to maintain separate code bases for each platform.

Up until now, React Native had limited support for the necessary APIs to perform cryptographic computations in
JavaScript. With the release of BigInt support however, it is now possible to build and deploy mobile apps that
communicate directly with the Internet Computer!

We have a starter template available for you here that uses agent-js (0.15) with no patches necessary! Check it out and
let us know what you think! Looking forward to seeing what you build.
